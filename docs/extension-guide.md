# 後續擴充指南

## 1. 新增百科條目

1. 開啟 `data/templates/entry-template.json`。
2. 複製欄位格式。
3. 將新條目加入 `data/entries.json` 陣列中。
4. 確認 `id` 不重複，建議格式：`g5-heat-transfer`、`j7-cell`。
5. 使用本機伺服器預覽，確認首頁、全部條目與詳細頁正常顯示。

## 2. 建議 id 命名規則

- 國小三年級：`g3-英文主題`
- 國小四年級：`g4-英文主題`
- 國小五年級：`g5-英文主題`
- 國小六年級：`g6-英文主題`
- 國中七年級：`j7-英文主題`
- 國中八年級：`j8-英文主題`
- 國中九年級：`j9-英文主題`

## 3. 條目品質檢查

- 科學概念是否正確？
- 是否符合指定年級？
- 是否包含角色對話？
- 是否有生活例子？
- 是否有迷思澄清？
- 小活動是否安全、材料易取得？
- 圖像提示詞是否能直接交給 AI 繪圖工具？
- 是否說明跨年級銜接？

## 4. 擴充到 100 篇以上的建議

- 每 10 篇條目檢查一次關鍵字一致性。
- 每個主題至少建立一篇基礎條目、一篇生活應用條目、一篇探究活動條目。
- 國中條目可增加模型、數據表、折線圖與推論問題。
- 國小條目以觀察、分類、操作與生活經驗為核心。
